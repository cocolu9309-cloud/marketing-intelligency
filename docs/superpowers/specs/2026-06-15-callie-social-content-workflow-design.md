# Callie 社媒内容生成 Workflow — 设计文档

## 1. 概述

**目标**：建立一套可复用的工作流，允许用户上传 Callie 产品链接 + 热门社媒视频，生成针对单一平台的内容包（英文贴文 + 11个Hashtag + 中文视频脚本 + 关键帧AI示意图）。

**核心输入**：
- 产品图片 + 产品链接 → AI Vision 提取产品结构化信息
- 热门社媒视频文件（mp4/mov）→ video-breakdown 提取视频结构
- 目标平台（TikTok / Instagram / Pinterest / YouTube Shorts 等）

**核心输出**：
- 英文贴文文案（平台定制）
- 11个 Hashtag
- 中文视频脚本（含分镜描述）
- 4-6帧关键帧 AI 示意图

---

## 2. 设计原则

1. **复用优先**：现有 `callie-social-posts` skill 和 `video-breakdown` skill 直接对接，不重复造轮子
2. **单平台输出**：每次只做一个平台，用户可多次运行覆盖不同平台
3. **步骤可干预**：每个主要步骤的结果展示给用户，用户可调整后继续
4. **AI 生图在最后**：关键帧示意图在所有文案确认后统一生成，不打断主流程

---

## 3. 工作流步骤

### Step 1：产品信息提取

**输入**：用户上传产品图片 + 粘贴 callie.com 产品链接

**处理**：
- AI Vision 分析产品图片，提取：产品名称、材质、风格、目标人群、适用节日/场景
- 结合链接中的产品标题和分类信息（用户粘贴文字描述）

**输出**：
```
产品名称: xxx
品类: 定制首饰/包袋/礼品
材质/风格: xxx
适用场景: 生日/纪念日/婚礼/母亲节...
目标人群: 情侣/朋友/家人/ bridesmaids...
```

**用户可调整**：是（若识别有误，用户可修正后继续）

---

### Step 2：热门视频结构拆解

**输入**：用户上传热门社媒视频文件（mp4/mov）

**处理**：
- 调用 video-breakdown skill 提取：
  - Hook 类型（悬念/情感/ POV / 礼物揭晓...）
  - CTA 类型（账号引导/购买引导/评论引导...）
  - 叙事结构（3段式/问题-解决/前后对比...）
  - 节奏分析（0-3秒/3-15秒/15-30秒各段内容）
  - 文字 Overlay 用法

**输出**：
```
Hook类型: 悬念留白 + 情感场景设定
CTA类型: 账号引导型（非直接购买）
叙事结构: 情感铺垫 → 悬念揭晓 → 账号沉淀
节奏分布:
  0-3秒: 悬念开头
  3-15秒: 产品/情感展示
  15-30秒: 揭晓 + 账号引导
文字Overlay: 少量关键词叠加
```

**用户可调整**：是（可上传另一支视频重新拆解）

---

### Step 3：内容生成

**输入**：Step 1 产品信息 + Step 2 视频结构 + 目标平台

**处理**：
- 加载 `references/brand-guide.md`（品牌定位/声调/禁忌）
- 加载 `references/platform-rules.md`（目标平台规则）
- 加载 `references/output-format.md`（输出格式）
- 合并 product insight + video insight，生成内容包

**输出（按平台）：**

TikTok 示例：
```
3秒Hook: [文字/画面描述]
15-30秒脚本: [逐段中文描述]
On-screen文字: [建议叠加的文字]
Sound建议: [类型建议，非具体音频]
英文Caption: [1-2句话]
11个Hashtag: [英文，平台适配]
```

Instagram Reels 示例：
```
Reel Hook: [3秒视觉开场]
Caption: [英文文案]
Carousel/Story idea: [如果适用]
5-8个Hashtag: [英文]
```

Pinterest 示例：
```
Pin标题: [英文，SEO友好]
Pin描述: [英文，1-2句话]
关键词: [5个核心词]
Board建议: [收藏夹名称]
视觉方向: [竖图构图描述]
```

**用户可调整**：是（可重新生成、更换平台、或修改产品/视频输入后重新生成）

---

### Step 4：关键帧示意图生成

**输入**：Step 3 确认的中文视频脚本

**处理**：
- 将脚本拆分为 4-6 个关键节点
- 对每个节点调用 AI 绘图（图像生成模型）生成参考示意图
- 配合文字分镜脚本（画面构图、镜头运动、情绪氛围）

**输出**：
```
关键帧1 [0-3秒]: [AI生成图] + [分镜描述]
关键帧2 [3-8秒]: [AI生成图] + [分镜描述]
关键帧3 [8-15秒]: [AI生成图] + [分镜描述]
关键帧4 [15-22秒]: [AI生成图] + [分镜描述]
关键帧5 [22-30秒]: [AI生成图] + [分镜描述]
```

**分镜脚本格式**：
```
[时间节点] 画面描述 | 镜头运动 | 文字Overlay | 情绪氛围
例: [0-3秒] 特写手捧礼盒，背景柔光 | 固定镜头，缓慢推进 | "She kept this for years" | 温暖期待
```

## 5. 各平台 Hashtag 规则

| 平台 | Hashtag 数量 | 规则 |
|------|-------------|------|
| TikTok | 11个 | 英文，品牌词 + 场景词 + 趋势词 |
| Instagram | 5-8个 | 英文，轻量 |
| Pinterest | 2-5个 | 英文，SEO优先 |
| YouTube Shorts | 5-8个 | 英文 + 少量中文拼音 |
| X | 0-2个 | 英文，非必须 |

---

## 6. 关键帧生成说明

- **图片比例**：9:16（竖版短视频）
- **风格**：温暖、现代、简洁，符合 Callie 品牌调性
- **避免**：文字写在图上（文字 Overlay 由剪辑时加）
- **输出格式**：图片文件 + 对应文字分镜脚本

---

## 7. 实施计划

### Phase 1：Skill 对接（不新增代码）
- 将 video-breakdown 输出格式与 callie-social-posts skill 输入对齐
- 更新 `callie-social-posts` skill 的 prompt，引导它接受 product insight + video insight 作为输入
- 不改变两个 skill 的内部逻辑

### Phase 2：Workflow 编排
- 在 superpowers 框架下创建新的 workflow
- 串联：用户输入 → product insight → video insight → content gen → keyframe gen
- 每个步骤输出中间结果供用户确认

### Phase 3：关键帧生图集成
- 确定 AI 生图工具（暂用 Mermaid 或 ASCII 分镜示意，AI 生图作为后续能力）
- 如已有图像生成 API，对接生成 9:16 关键帧参考图

---

## 8. 已知限制

- video-breakdown skill 尚未验证远程推送状态，部署时需确认
- AI 生图能力待集成，当前 Phase 1-2 可用文字分镜替代
- 产品信息依赖用户上传图片质量，极模糊图片可能影响识别准确率
