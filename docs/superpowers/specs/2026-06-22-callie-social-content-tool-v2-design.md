# Callie 社媒内容生成工具 v2 — 设计文档

**日期**: 2026-06-22
**状态**: 已确认

---

## 一、背景与目标

### 现状
现有 `callie-social-content-workflow` v1 已完成基础功能（FastAPI 网页工具 + 视频帧提取 + AI 文案生成 + Excel 输出），但存在以下改进空间：
- 热门内容分析维度不够完整
- 无自动产品匹配推荐功能
- 设计对接文档未按原始格式生成
- 跨环境部署仍有配置复杂度问题

### 目标
设计 v2 版本，实现**自然流社媒内容生成工具**：
1. 用户上传 TikTok/Instagram 热门贴文（图片/视频）
2. AI 拆解分析内容（Hook/分类/主干/信息密度/CTA/配色/主标题/元素占比/内容类型）
3. 自动从 Callie 产品库匹配推荐最适合的 Callie 产品（含上架图片下载）
4. 生成两个 Excel：内容生成包 + 设计对接文档（含 6 帧参考示意图）
5. 支持不同运营、不同电脑、不同 AI Agent 环境使用

## 二、技术架构

### 2.1 技术栈

| 模块 | 技术选型 | 说明 |
|------|----------|------|
| 前端 | 单页 HTML + 原生 JS | 无框架，浏览器直接打开 |
| 后端 | Python FastAPI | 单文件 `app.py` |
| AI 视觉分析 | SiliconFlow `Qwen/Qwen3-VL-8B-Instruct` | 多模态，理解图片/视频帧 |
| AI 生图 | SiliconFlow `Qwen/Qwen-Image-Edit-2509` | 关键帧参考图生成 |
| 视频抽帧 | 混合模式 | OpenCV 候选帧 + AI 精选 6 帧 |
| Excel 生成 | openpyxl | 输出格式化 Excel |
| 产品库 | 本地 SQLite | 定期爬虫更新 |

### 2.2 整体流程

```
用户上传热门图片/视频
    ↓
前端 JS 解析视频 → OpenCV 检测场景变化点（10-15 个候选帧）
    ↓
AI 视觉模型从候选帧中精选 6 帧
    ↓
AI 分析 6 帧 + 综合分析（Hook/分类/主干/信息密度/CTA/配色/主标题/元素占比/内容类型）
    ↓
从本地产品库综合评分匹配 → 推荐 Top 3 产品
    ↓
自动下载 Top 3 产品的上架图片（白底产品图）
    ↓
综合输出：
├── 内容生成包.xlsx（英文文案 + 11条Hashtag + 设计底层逻辑）
└── 设计对接文档.xlsx（完整设计参数 + 6帧参考示意图，含产品图 + Overlay英文文案）
```

### 2.3 部署结构

```
Callie社媒内容工具/
├── start.bat              ← 双击启动（创建虚拟环境 + 启动服务 + 打开浏览器）
├── stop.bat               ← 停止服务
├── config.yaml            ← 管理员预置 API Key（运营不可改）
├── update.bat             ← 更新产品库（触发爬虫）
├── app.py                 ← FastAPI 后端
├── index.html             ← 前端页面
├── requirements.txt       ← Python 依赖
├── scripts/
│   └── crawl_products.py  ← 产品爬虫脚本
└── data/
    └── products.db        ← 本地产品库（首次运行自动创建）
```

### 2.4 启动流程（运营操作）

1. 双击 `start.bat`
2. 自动创建 Python 虚拟环境 + 安装依赖
3. 自动打开浏览器 `http://localhost:8000`
4. 运营上传热门内容，开始使用

### 2.5 跨环境使用
- 管理员将分发包复制到新电脑
- 确保新电脑有网络（下载依赖）
- 无需手动配置，双击即用
- API Key 由管理员统一配置在 `config.yaml`，运营不可改

## 三、AI 视觉分析维度

### 3.1 图片/视频帧分析项（9项）

| # | 分析项 | 说明 |
|---|--------|------|
| 1 | Hook 元素 | 开头如何吸引注意力（悬念/情感/POV/礼物揭晓/问题抛出/前后对比） |
| 2 | 内容分类 | 产品展示/用户评价/开箱/教程/品牌故事等 |
| 3 | 主干内容摘要 | 视频/图片的核心信息 |
| 4 | 信息密度 | 高/中/低 |
| 5 | CTA | 行动号召类型和位置 |
| 6 | 配色风格 | 整体色调和配色方案 |
| 7 | 主标题/副标题/主要元素 | 图片中的文字和核心视觉元素 |
| 8 | 各元素占比 | 图片中各信息元素的面积比例分布 |
| 9 | 内容类型 | 促销/品牌故事/产品演示等 |

### 3.2 产品推荐匹配逻辑（综合评分）

**三维度打分**：
1. **标签匹配**：内容标签（如：宠物、庆祝、生日）→ 产品标签（宠物纪念品、生日礼物）
2. **场景匹配**：内容场景关键词 → 产品适用场景字段（母亲节送礼、婚礼纪念等）
3. **视觉风格匹配**：色调/材质/风格 → 产品风格/材质属性（木质、温馨、暖色调）

→ 综合三个维度打分，推荐 Top 3 产品

## 四、产品库设计

### 4.1 数据来源
Callie.com 官方爬虫（`crawl_products.py`）

### 4.2 更新机制
- **首次使用**：管理员运行爬虫脚本，自动抓取 Callie.com 全站产品 → 存入 `products.db`
- **日常使用**：工具从本地 SQLite 读取，秒速匹配
- **按需更新**：管理员手动点击"刷新产品库"按钮，或运行 `update.bat`

### 4.3 产品信息字段

| 字段 | 说明 |
|------|------|
| 产品名称（英文） | Callie 官网产品标题 |
| 产品名称（中文） | AI 翻译 + 人工校正 |
| 产品分类 | 按 Callie 导航栏分类（个性化礼品/宠物纪念品/家居装饰等） |
| 产品标签 | AI 提取（场景/风格/材质/受众） |
| 产品图片 URL | 供 AI 分析和匹配参考 |
| 产品描述 | 官网英文描述 |
| 适用场景 | 生日/婚礼/纪念日/节日等 |
| 上架状态 | 在售/缺货/停售 |

## 六、关键技术决策

### 6.1 视频智能抽帧：混合模式
1. **OpenCV 场景检测**：检测视频中画面大幅变化点，定位 10-15 个候选帧位置
2. **AI 精选**：将 10-15 个候选帧逐帧分析，AI 挑选最重要的 6 帧

**优势**：平衡速度和准确性，避免纯算法误判重要情感画面

### 6.2 关键帧参考图生成
使用 SiliconFlow `Qwen/Qwen-Image-Edit-2509` 生成 6 帧参考示意图：
- 参考元素：热门内容的构图 + Callie 产品实物图 + 品牌风格
- Overlay 文案嵌入画面（英文）
- 图片嵌入设计对接文档 Excel

### 6.3 产品图片下载
匹配到 Top 3 产品后，自动下载对应产品的上架图片（白底图），用于：
- AI 生成参考示意图时合成产品图
- 设计对接文档中的产品展示
